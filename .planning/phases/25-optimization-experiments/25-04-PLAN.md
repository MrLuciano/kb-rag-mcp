      - `__init__(self, vector_store: Any, dataset: GoldenDataset, output_dir: Path = Path("data/experiments"))` — initializes `ExperimentResultStore`, stores vector_store and dataset
      - `async def run_chunking_experiment(self, docs_path: Path, strategy: str, chunk_size: Optional[int] = None, overlap: Optional[int] = None, top_k: int = 5, run_id: Optional[str] = None) -> dict[str, Any]`:
        1. Generate `run_id` if not provided: `f"chunk_{strategy}_{chunk_size}_{datetime.now().strftime('%Y%m%d_%H%M%S')}"`
        2. Create strategy via `create_strategy(strategy, chunk_size=chunk_size, overlap=overlap)`
        3. Create `ChunkingEngine(strategy, self.vector_store, self.dataset)`
        4. Run `engine.run_experiment(docs_path, top_k=top_k, clean=True)`
        5. Save results via `self.store.save(run_id, params={...}, metrics={...}, strategy=strategy, variant="chunking")`
        6. Return full result dict including `run_id` and `output_path`
      - `async def run_scoring_experiment(self, variant: str, top_k: int = 5, run_id: Optional[str] = None, **variant_kwargs) -> dict[str, Any]`:
        1. Generate `run_id` if not provided: `f"score_{variant}_{datetime.now().strftime('%Y%m%d_%H%M%S')}"`
        2. Create variant via `create_variant(variant, **variant_kwargs)`
        3. Create `ScoringEngine(variant, self.vector_store, self.dataset)`
        4. Run `engine.run_experiment(top_k=top_k)`
        5. Save results via `self.store.save(run_id, params={...}, metrics={...}, strategy=variant, variant="scoring")`
        6. Return full result dict including `run_id` and `output_path`
      - `async def run_parameter_sweep(self, experiment_type: str, param_grid: dict, docs_path: Optional[Path] = None) -> list[dict[str, Any]]`:
        1. Accepts `experiment_type`: "chunking" or "scoring"
        2. `param_grid` is a dict of parameter names to lists of values (e.g., `{"strategy": ["fixed", "recursive"], "chunk_size": [256, 512]}`)
        3. Generate all combinations via `itertools.product`
        4. Run one experiment per combination, collecting results
        5. Return list of result dicts
        6. Auto-save baseline for the first run if no baseline exists yet
      - `async def compare_runs(self, run_ids: list[str]) -> dict[str, Any]`:
        1. Delegates to `self.store.compare(run_ids)`
        2. Adds a formatted console table string using `ResultsExporter.to_console()` pattern
        3. Returns comparison dict with `table` (string) and `deltas` (dict)
