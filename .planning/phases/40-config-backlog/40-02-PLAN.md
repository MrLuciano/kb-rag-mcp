## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP client → Config API (health server) | Untrusted internal clients can read/write config values |
| ConfigLoader → SQLite | Internal data store; WAL mode enabled per D-14 |

| server.py → ConfigLoader | Server config now flows through centralized loader |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
