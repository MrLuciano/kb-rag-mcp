# Plan 22-01: Integration Checker CI Gate

## Objective

Create a Python-based integration checker script that validates cross-referencing between
documentation, requirement traceability, and plan implementation; wire it into GitHub Actions
as a new CI job that runs after tests and fails on unresolved gaps.

## Tasks

### Task 1: Create `scripts/check-integration-gaps.py`

Create a standalone Python script that performs three gap checks:

**Check 1 — VERIFICATION.md presence (D-01):**
- Scan `.planning/phases/*/` directories
- Each directory should contain a `*VERIFICATION*` file
- Report phases missing verification docs

**Check 2 — REQUIREMENTS.md traceability (D-02):**
- Parse `.planning/REQUIREMENTS.md` for the traceability table
- Each REQ-ID row with status != "Complete" counts as gated
- Every REQ-ID must have: a phase mapping, a plan that addresses it
- Cross-reference against plan SUMMARY.md files to confirm each REQ-ID was completed
- Flag any REQ-IDs that are "🔄 Planned" or otherwise not complete

**Check 3 — SUMMARY.md file references (D-03):**
- Parse all `*SUMMARY.md` files across `.planning/phases/*/`
- Extract `key-files.modified` and `key-files.created` from frontmatter
- Verify every referenced file path exists on disk
- Report stale references

**Output format (D-09):**
- Human-readable: Rich table with columns (Check | Status | Details)
- Machine-parseable: JSON summary file written to `scripts/check-integration-gaps-results.json`
- JSON contains: timestamp, per-check pass/fail, list of all gaps, exit_code

**Exit code (D-08):**
- Exit 0 if all gaps pass
- Exit 1 if any gap fails — CI will catch this

**Pattern:** Follow `scripts/logging-audit.py` (argparse, no external imports beyond stdlib + Rich).

### Task 2: Update `.github/workflows/ci.yml`

Add a new `integration-check` job:

```yaml
integration-check:
    runs-on: ubuntu-latest
    needs: test
    steps:
      - uses: actions/checkout@v4
      - name: Set up Python
        uses: actions/setup-python@v5
        with:
          python-version: "3.11"
      - name: Install dependencies
        run: |
          python -m pip install --upgrade pip
          pip install -r requirements.txt
      - name: Run integration gap checker
        run: |
          python3 scripts/check-integration-gaps.py
```

Place it after the `english-audit` job and before `helm-lint` (keep alphabetical grouping).
The `needs: test` ensures it runs after tests (CICHECK-01). It validates gaps (CICHECK-02),
fails CI on gaps (CICHECK-03), and reports results via stdout (CICHECK-04).
