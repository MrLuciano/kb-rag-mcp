### Task 3: Security Audit

<read_first>
- kb_server/auth/router.py
- kb_server/auth/middleware.py
- kb_server/auth/models.py
- kb_server/server.py
</read_first>

**Description:** Review auth infrastructure for security gaps.

**Review areas:**
- API key hashing (SHA-256) and session token entropy
- Session cookie flags: Secure, HttpOnly, SameSite, max-age
- Rate limiting on login/verify_key endpoints
- PII exposure in user list, audit log, GDPR erasure
- SQL injection vectors in config values and search queries

**Acceptance criteria:**
- Audit report produced in `.planning/reports/security-audit-v0.1.5.md`
- 0 critical or high severity findings
- Any findings have remediation plan or explicit deferral
