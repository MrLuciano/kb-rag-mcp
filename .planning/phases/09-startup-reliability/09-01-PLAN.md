<objective>
Harden and verify cross-encoder model lazy loading so the server starts without loading
the ~500MB cross-encoder model. The model should only load on the first `predict()` call
inside `rerank()`.

**Purpose:** Eliminate ~10s startup latency and ~500MB memory overhead caused by loading
the cross-encoder at import time. The lazy loading pattern already exists in
`CrossEncoderReranker._load_model()` — this plan adds explicit regression tests and a
startup log to ensure the invariant is maintained.

**Output:** Hardened lazy loading + regression tests that catch accidental module-level
import of `sentence_transformers.CrossEncoder`.
</objective>
