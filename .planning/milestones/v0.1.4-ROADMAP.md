# Milestone v0.1.4: Platform, Analytics & Enterprise

**Status:** ✅ SHIPPED 2026-06-11

## Overview

Transformed kb-rag-mcp from a single-collection RAG server into a multi-tenant, enterprise-ready platform with streaming HTTP transport, knowledge graph, authentication, rate limiting, quotas, circuit breakers, and retrieval caching.

## Milestone Summary

**Key Decisions:**

- Local embedding model (LM Studio/Ollama) for data sovereignty
- MCP protocol for AI tool integration (Claude, Cursor, OpenCode, Copilot)
- SQLite config table with hot-reload for all env vars
- Passwordless API key + JWT session auth
- fastembed BM25 for sparse vectors (embedded, no separate server)

**Issues Resolved:**

- Context overflow at 100+ phases (resolved by ROADMAP.md split)
- Phase insertion confusion (resolved by decimal phase numbering)
- Grafana datasource error (stable UID)
- MagicMock pollution in test suite

**Issues Deferred:**

- Nyquist validation for 12 phases (not required at time of shipping)
- `datetime.utcnow()` deprecation warning (info severity only)
- Dedicated test file for multi-KB search (functional coverage complete in 4 existing files)

**Technical Debt Incurred:**

- `PayloadSchemaType` enum assertion weakened in test_payload_indexes.py
- Some workflows still have hardcoded paths
- `datetime.utcnow()` deprecated in Python 3.14 (warning only)
