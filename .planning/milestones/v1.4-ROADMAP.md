# Milestone v1.4: Platform, Analytics & Enterprise

**Status:** ✅ SHIPPED 2026-06-11
**Phases:** 23-37 (Phases 24, 25 deferred)
**Total Plans:** 21

## Overview

Enterprise hardening of kb-rag-mcp: enterprise data source connectors (Confluence, JIRA, Git), cross-document knowledge graph, MCP prompt templates, API key authentication, request rate limiting, upload/index quotas, multi-KB aggregated search, provider budget & circuit breaker, and request-level retrieval cache. Documentation overhaul, KB content discoverability, KB registry, and MCP Streamable HTTP transport also completed.

## Milestone Summary

**Deferred Phases:**
- Phase 24: RAGAS Evaluation Pipeline (3 plans deferred to backlog)
- Phase 25: Optimization Experiments (2 plans deferred to backlog)

**Key Decisions:**
- Connector factory pattern with eager import registration (ingest/connectors/__init__.py)
- Per-KB graph metadata (not global)
- Template-only prompts (no LLM backend integration for prompt rendering)
- SHA-256 hashed API keys with revocation (no plaintext storage)
- In-memory token bucket rate limiting (single instance, no Redis needed)
- Cumulative quota counters with explicit reset (no time-period expiry)
- Same-dimension assumption for multi-KB search (validated at query time)
- In-memory circuit breaker state (resets on restart)
- Cache structured results before rendering (not rendered output)
