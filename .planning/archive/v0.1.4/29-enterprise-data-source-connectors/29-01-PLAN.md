## Confluence 7.9.3 (Server/Data Center)

All Confluence connector tasks support both Cloud and 7.9.3 variants:

- **Auth 7.9.3:** Basic auth (`username:password` base64) or personal access tokens
- **Auth Cloud:** OAuth 2.0 client credentials
- **API 7.9.3:** `/rest/api/content` with `limit` (max 200) and `start` offset pagination — no cursor
- **API Cloud:** Same endpoint with cursor-based pagination
- **CQL 7.9.3:** `space=<key>`, `type=page`, `lastModified` date filtering work identically
- **Markup:** 7.9.3 uses Storage Format (XHTML-based); Cloud uses ADF with Storage Format fallback
</confluence_793>
